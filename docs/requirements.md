# プロフェッショナル・タスクボード — 要件定義書

## 1. プロダクト概要

モダンでクリーンなデザインの高度なタスク管理Webアプリケーション。
複数プロジェクトの管理、カンバンボード、ガントチャート、分析ダッシュボードを備え、チームのタスク進捗を一元管理する。

## 2. 技術スタック

| カテゴリ | 技術 |
|---|---|
| フレームワーク | React 19 + TypeScript 5.8 |
| ビルドツール | Vite 6 |
| スタイリング | Tailwind CSS v4 |
| アニメーション | Motion (framer-motion) |
| グラフ | Recharts |
| 日付処理 | date-fns（日本語ロケール対応） |
| アイコン | Lucide React |
| バックエンド（未実装） | Express + better-sqlite3 |
| AI連携（未実装） | @google/genai |

## 3. データモデル

### 3.1 Task（タスク）
| フィールド | 型 | 説明 |
|---|---|---|
| id | string | 一意識別子 |
| title | string | タスク名 |
| startDate | string (YYYY-MM-DD) | 開始日 |
| dueDate | string (YYYY-MM-DD) | 期限日 |
| priority | Priority (高/中/低) | 優先度 |
| status | Status (未着手/進行中/レビュー待ち/完了) | ステータス |
| memo | string | メモ・詳細 |
| assigneeIds | string[] | 担当者IDリスト（複数割当可） |
| columnId | string | 所属カラムID |
| projectId | string | 所属プロジェクトID |

### 3.2 Project（プロジェクト）
| フィールド | 型 | 説明 |
|---|---|---|
| id | string | 一意識別子 |
| name | string | プロジェクト名 |
| color | string (hex) | テーマカラー |
| notes | string (optional) | 共有事項メモ |

### 3.3 Column（カラム/リスト）
| フィールド | 型 | 説明 |
|---|---|---|
| id | string | 一意識別子 |
| title | string | カラム名 |
| projectId | string | 所属プロジェクトID |
| order | number | 表示順序 |
| color | string (optional, hex) | ヘッダーカラー |

### 3.4 Assignee（担当者）
| フィールド | 型 | 説明 |
|---|---|---|
| id | string | 一意識別子 |
| name | string | 氏名 |
| avatar | string | アバター表示テキスト（姓） |
| color | string (hex) | アバターカラー |

## 4. 機能要件

### 4.1 認証
- **FR-AUTH-01**: ログイン画面でID/パスワード入力によるログイン
- **FR-AUTH-02**: ログアウト機能（サイドバー・設定画面から実行可能）
- **FR-AUTH-03**: ログインエラー時のエラーメッセージ表示

### 4.2 ダッシュボード
- **FR-DASH-01**: 統計カード表示（全タスク数、期限切れ数、完了済み数、プロジェクト数）
- **FR-DASH-02**: ステータス別タスク分布（ドーナツチャート）
- **FR-DASH-03**: 優先度別タスク分布（棒グラフ）
- **FR-DASH-04**: 最近のタスク一覧（直近5件、期限日降順）
- **FR-DASH-05**: プロジェクト一覧と進捗率プログレスバー
- **FR-DASH-06**: プロジェクトクリックでボードビューへ遷移

### 4.3 ボードビュー（カンバン）
- **FR-BOARD-01**: プロジェクトごとのカラム（リスト）表示
- **FR-BOARD-02**: カラム内タスクカード表示（優先度・ステータスバッジ、期限、担当者アバター、メモアイコン）
- **FR-BOARD-03**: タスクのドラッグ＆ドロップによるカラム間移動
- **FR-BOARD-04**: ドロップターゲットのハイライト表示
- **FR-BOARD-05**: カラム追加・名前変更・削除
- **FR-BOARD-06**: カラム内タスクの優先度ソート（昇順/降順/解除の3段階トグル）
- **FR-BOARD-07**: タスク追加ボタン（カラムごと）
- **FR-BOARD-08**: 期限切れタスクの赤枠ハイライト
- **FR-BOARD-09**: タスクカードのカラー選択付き編集

### 4.4 ガントチャート
- **FR-GANTT-01**: タスクを開始日順にソートした横棒タイムライン表示
- **FR-GANTT-02**: 31日間の日付ヘッダー（月曜始まり）
- **FR-GANTT-03**: 今日のハイライト表示（カラムとインジケーターライン）
- **FR-GANTT-04**: 前週/次週/今日ボタンによるナビゲーション
- **FR-GANTT-05**: タスクバーの色分け（完了: 緑、それ以外: 青）
- **FR-GANTT-06**: タスクバー・タスク名クリックで編集モーダル表示
- **FR-GANTT-07**: 担当者アバターの行内表示

### 4.5 分析ビュー
- **FR-ANLY-01**: 担当者別タスク割り当て率ドーナツチャート
- **FR-ANLY-02**: 担当者別タスク遂行率横棒グラフ
- **FR-ANLY-03**: 担当者別詳細テーブル（割り当て数、完了数、割り当て率、遂行率）
- **FR-ANLY-04**: テーブル内プログレスバー表示

### 4.6 タスク管理（CRUD）
- **FR-TASK-01**: タスク作成（カラム指定、デフォルト値自動設定）
- **FR-TASK-02**: タスク編集モーダル（タスク名、開始日、期限日、プロジェクト、カラム、優先度、ステータス、担当者、メモ）
- **FR-TASK-03**: タスク削除（カード上のXボタン、またはモーダル内の削除リンク）
- **FR-TASK-04**: 複数担当者のトグル割り当て

### 4.7 プロジェクト管理
- **FR-PROJ-01**: プロジェクト追加（名前・カラー指定）
- **FR-PROJ-02**: プロジェクト削除（最低1件は残す制約、関連タスクも連動削除）
- **FR-PROJ-03**: サイドバーでのプロジェクト切り替え
- **FR-PROJ-04**: ヘッダーのプロジェクトドロップダウンメニューによる切り替え
- **FR-PROJ-05**: プロジェクトノート（共有事項）のサブヘッダー表示

### 4.8 担当者管理
- **FR-ASGN-01**: 担当者追加（名前・カラー指定）
- **FR-ASGN-02**: 担当者編集（名前・カラー変更）
- **FR-ASGN-03**: 担当者削除（関連タスクからも自動除去）
- **FR-ASGN-04**: 担当者によるタスクフィルタリング（トグル選択）
- **FR-ASGN-05**: フィルター解除リンク

### 4.9 検索
- **FR-SRCH-01**: タスク名・メモによるリアルタイム検索
- **FR-SRCH-02**: 検索クエリクリアボタン
- **FR-SRCH-03**: 検索結果が0件の場合の空状態表示

### 4.10 警告通知
- **FR-WARN-01**: 未割当タスク警告ポップアップ（右下、アンバー色）
- **FR-WARN-02**: 期限間近警告ポップアップ（3日以内、青色）
- **FR-WARN-03**: 期限超過警告ポップアップ（赤色）
- **FR-WARN-04**: 各ポップアップの個別閉じるボタン
- **FR-WARN-05**: アニメーション付きスライドイン/アウト

### 4.11 設定画面
- **FR-SET-01**: プロフィール編集（ユーザー名、表示名、メールアドレス、アバター）
- **FR-SET-02**: パスワード変更（現在/新規/確認の3フィールド）
- **FR-SET-03**: セキュリティ設定（2段階認証トグル、ログイン履歴表示）
- **FR-SET-04**: 通知設定（プッシュ通知、メール通知、週次レポートのON/OFF）
- **FR-SET-05**: 外観設定（ライト/ダーク/システムテーマ切り替え、テーマカラー選択）
- **FR-SET-06**: 言語・地域設定（表示言語、タイムゾーン選択）

## 5. 非機能要件

### 5.1 UI/UX
- **NFR-UI-01**: レスポンシブデザイン（モバイル・デスクトップ対応）
- **NFR-UI-02**: ダークモード完全対応
- **NFR-UI-03**: スムーズなアニメーション・トランジション（Motion使用）
- **NFR-UI-04**: Inter フォント使用
- **NFR-UI-05**: カスタムスクロールバー（WebKit・Firefox対応）
- **NFR-UI-06**: エラーバウンダリによるクラッシュ時のフォールバック画面

### 5.2 パフォーマンス
- **NFR-PERF-01**: useMemoによるフィルタリング・ソート結果のメモ化
- **NFR-PERF-02**: AnimatePresenceによる効率的なアニメーション管理

### 5.3 永続化
- **NFR-PERS-01**: テーマ設定のlocalStorage永続化
- **NFR-PERS-02**: テーマカラーのlocalStorage永続化

## 6. 現在の制約・未実装事項

| 項目 | 状態 | 説明 |
|---|---|---|
| 認証 | モック | admin/admin ハードコード |
| データ永続化 | 未実装 | インメモリのみ、リロードで消失 |
| バックエンドAPI | 未実装 | Express + SQLite の依存あるが未使用 |
| AI連携 | 未実装 | @google/genai の依存あるが未使用 |
| 設定保存 | 一部のみ | テーマとテーマカラーのみlocalStorage保存 |
| カラム内タスク並べ替え | 未実装 | カラム間移動のみ可能 |
| ガントチャートのインタラクション | 未実装 | 表示のみ（ドラッグでの期間変更不可） |
| プロフィール・パスワード変更 | UIのみ | 実際の保存処理なし |
| 通知 | UIのみ | 実際のプッシュ/メール通知なし |
| リアルタイム同期 | 未実装 | マルチユーザー対応なし |
